Notes

  • The calculation has following restrictions:
    • MAFFT
      • length of sequences: 30000
      • number of sequences: 20000
    • ClustalW2
      • length of sequences: 30000
      • number of sequences: 1000
    • No alignment
      • length of sequences: 30000
      • number of sequences: 20000
  • The maximum file size is 10 MB.
  • Results are stored for 7 days.
